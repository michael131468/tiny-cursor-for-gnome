# Tiny Cursor for Gnome

Basically an invisible cursor.

Useful for when you're using something like a KVM and want to avoid motion
sickness with the cursor lag. This hides the cursor so you can use your native
cursor on the viewer machine. It leaves enough of a dot behind though for
debugging in case the cursor is not aligned correctly.

## install.sh

Some basic instructions on how to install and use on a gnome system.
